# Morning Briefing Skill

## When to use
- App opens fresh in morning
- "What is my day today?"
- "Good morning"
- "Brief me"

## Workflow
1. Get weather for user's city
2. Get news by user interests
3. Load today's tasks and appointments
4. Suggest quick breakfast
5. List shopping needs
6. Deliver formatted briefing
